Feedback Loop — Маршрутизация при fail

Когда HM-5 (Контролёр) обнаруживает fail по метрикам, Feedback Router определяет КУДА вернуть текст:

→ HM-2 (Архитектор): если fail по para_cv, section_cv, triplets, connector density
→ HM-3 (Лексикограф): если fail по P7 violations, modal hedging, attribution
→ HM-4 (Голос): если fail по burstiness, perplexity lift, But:However ratio, imperfection texture

Максимум 2 итерации feedback loop. После 2-го fail — отправка CEO Agent с report для ручного решения.

Общая статистика архитектуры v2

CEO: 1 агент
Manager: 2 агента (Content + Humanizer)
Micro Manager: 12 агентов (7 Content + 5 Humanizer)
Worker: 50 агентов (30 Content + 20 Humanizer)
Shared Service: 4 сервиса (Language Gate, Domain Gate, SourceFinder, DOCX Formatter)

ИТОГО: 65 агентов + 4 shared services = 69 компонентов

Правило коммуникации: Worker ↑ Micro Manager ↑ Manager ↑ CEO.
Горизонтальная связь Workers — только через Micro Manager.
Shared Services — доступны всем агентам своего стрима.
