→ HM-2: fail по para_cv, section_cv, triplets, connector density, scaffold signals (P44–P53)

Data Flow — End-to-End (VKR бакалавр, экономика, RU)

1. CEO получает params: stream_id=vkr, topic, domain=economics, language=ru, level=bachelor
2. CEO → Content Manager → MM-3 (ВКР)
3. MM-3 → SourceFinder Service ① → source_list (≥50, ГОСТ). API Gateway обслуживает LLM-вызов
4. MM-3 последовательно через API Gateway: Тит. лист → Аннотация → TOC (placeholder) → Введение → Гл.1 → Гл.2 → Гл.3 → Заключение
5. MM-3 обновляет TOC (2nd pass)
6. → Content QA Gate (Pattern Scanner SS): structural check + viz count ≥ 5. Fail → re-gen
7. → Handoff Gate: text + economics + academic + ru + sources + viz_count
8. HM-1: Pattern Scanner (SS) → baseline. Metrics Engine. Domain Classifier → Domain Gate (economics). Language Gate: AC-15 skip_perplexity=true
9. HM-2: Scaffold Breaker (P9,P12,P25–27,P44–53), Format Cleaner (P13–18, F1–F4b), Connector Tuner, Triplet Buster. Chunks via SS. API via Gateway
10. → [QA Coherence] cosine similarity ≥ 0.85? OK →
11. HM-3: Vocab Eliminator (P7+P28+P8+P11+F8, domain exemption via Domain Gate), Hedging Auditor, Attribution Fixer (ГОСТ [N] via Language Gate AC-2), Result Formatter
12. → [QA Coherence + Register] drift check. OK →
13. HM-4: Soul Injector (P30+F5/F6/F7, Language Gate: AC-5 skip figurative, AC-4 passive 70%, AC-7 skip F5/F6 for RU academic), Rhythm Shaper, Pass B Auditor, Example Loader
14. → [QA Coherence] post-voice-injection check (over-imperfection guard). OK →
15. HM-5: Score Engine (Pattern Scanner SS rescan, 20 metrics), Detector Probe (opt), Feedback Router. Fail → HM-2/3/4 (max 2). Pass → Report Builder
16. CEO → DOCX Formatter (SS): ГОСТ 7.32-2017, Times New Roman 14pt, 1.5, tables + figures
17. Output: .txt + .docx + score_report.json + score_report.txt
